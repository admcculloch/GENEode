1. Please ensure that your local machine has installed anaconda and pip(pip is pre-installed in unlx-like systems). Anaconda has integraded many useful packages already.

https://docs.anaconda.com/anaconda/install/

Our implementation is based on Python3. When you install anaconda, python is automatically installed on version 3.7.4.

2. Please install the following packages in your local environment.
pip install chaospy
pip install uncertainpy
